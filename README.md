# QWA application 
Web applization for quiz making/solving.
Application allows signed up users to make, edit, publish quizzes.
Unregistered users are allow to solve quizzes.


## Technologies
*Java 
*Spring Boot
*Hibernate
*MySQL
*Angular 6
*Bootstrap
*Angular Materials
